exibe informações sobre as colunas em uma determinada tabela. Também funciona para visualizações. SHOW COLUMNS exibe informações apenas para as colunas para as quais você tem algum privilégio.

Sintaxe:

SHOW COLUMNS FROM nome_tabela;

Exemplos:

SHOW COLUMNS FROM tblaluno

Resultado:

Field Type Null Key Default Extra
codaluno int(11) YES
nome varchar(50) YES
endereco varchar(50) YES
telefone varchar(50) YES
nascimento date YES
sexo varchar(1) YES
observacao varchar(100) YES
mãe varchar(60) YES
